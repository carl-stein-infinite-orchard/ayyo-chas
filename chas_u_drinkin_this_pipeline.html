Daily pipeline
Auto

Manual step
01 · Nanobanana — flavor oracle
Auto
Claude API call generates one flavor per day: picks a category (interesting / gross / impossible), writes the flavor name, a fake tagline, a mock ingredients list, and a short "vibe brief" for the image generator. Outputs a brand_identity.json.
Claude API cron / n8n brand_identity.json
Emits
brand_identity.json · flavor name, category, tagline, palette hint, vibe brief, fake ingredients
02 · Can art generation
Auto
brand_identity.json feeds a structured DALL-E 3 or Flux prompt. Generates a flat label illustration (no 3D — flat label art is easier to auto-composite). A second pass composites the label onto a clean seltzer can mockup template via a Pillow/Canvas script.
DALL-E 3 / Flux Python Pillow can mockup template
Emits
can_front.png · label_raw.png · label_composited.png
03 · You review the can
Review
A simple approval dashboard (or just a Slack/email preview) shows the can art and the flavor identity card. You approve, regenerate, or tweak the prompt. This is the creative gate — the one human moment before video production starts.
approval UI / Slack regenerate hook
Gate
approved_can.png → triggers Remotion render
04 · Remotion animation
Auto
A parametric Remotion template reads brand_identity.json and the approved can PNG. Animates: (1) cold open with category badge (Gross / Interesting / Impossible), (2) flavor name reveal with kinetic type, (3) can rotates and settles into frame with a held end card. Target: 20–30s for Reels.
Remotion Monty agent brand_identity.json
Emits
video_silent.mp4 · thumbnail.jpg
05 · Audio layer
Manual
Two modes: (A) royalty-free track auto-selected from a curated pool via ElevenLabs sound effects + Suno-generated backing — fully automated. (B) You swap in a trending audio clip manually in CapCut before posting — this is how you chase the algorithm. Mode B is always worth doing when you spot a moment.
ElevenLabs SFX Suno CapCut (manual swap)
Emits
video_final.mp4 · ready_to_post flag
06 · Caption + post
Review
Claude generates 3 caption options (one dry, one chaotic, one engagement-baiting). You pick or edit. Auto-post via Instagram Graph API + Twitter API v2, or paste manually. Instagram Reels thumbnail is set to the held end-card frame.
Claude API Instagram Graph API Twitter API v2
Emits
posted · engagement tracking begins
brand_identity.json schema
"flavor": "Gym Sock & Lime", "category": "gross", "tagline": "refreshingly horrifying", "fake_ingredients": ["carbonated water", "natural flavors", "citric acid", "athlete's foot extract*"], "palette": { "primary": "#A3C96E", "accent": "#F2E14B", "vibe": "sports-brand-gone-wrong" }, "image_prompt": "flat label art, athletic green with a cartoon worn gym sock and lime wedge, bold retro typography, slightly disgusting but charming", "volume_oz": 12, "limited_edition": false, "disclaimer": "*not a real ingredient. do not drink."
Key decisions + tradeoffs
Decision Chosen approach Risk
Can 3D mockup Flat label composite onto static PNG template — simpler to automate than Blender 3D renders Low
Trending audio Manual swap in CapCut — Instagram blocks programmatic trending audio injection Low
Image gen model DALL-E 3 for speed + API simplicity; Flux as upgrade path for style consistency Med
Remotion template Single parametric template; brand_identity.json drives all copy + colors Low
Instagram posting Graph API requires a Facebook-linked Business account — needs setup if not already done Med
Content archive All generated assets stored in orchard/projects/chas-u-drinkin-this/YYYY-MM-DD/ Low
What you build first
  • Nanobanana Claude API prompt + JSON schema
  • brand_identity.json → DALL-E 3 pipeline
  • Pillow can composite script
  • Approval UI (even a basic HTML preview)
  • Remotion parametric template
  • Caption generator + manual post flow
What you automate later
  • Instagram Graph API auto-post
  • Twitter/X API v2 auto-post
  • Suno background music generation
  • Engagement analytics dashboard
  • Flavor archive / hall of fame page
  • "Vote on tomorrow's flavor" Stories poll
